# Patrumin Website Redesign Brief

## Objective
Redesign the current Patrumin Investors website into a modern, institutional, boutique RIA website.

The target aesthetic should merge the institutional polish of Morgan Stanley, Goldman Sachs, J.P. Morgan, and BlackRock, while preserving Patrumin’s own brand identity, existing color scheme, and current website content where appropriate.

Do not copy any proprietary layout, text, images, trademarks, or exact design from those firms. Use them only as high-level inspiration for tone, hierarchy, spacing, navigation structure, and institutional credibility.

## Brand Positioning
Patrumin should feel like:
- boutique but institutional
- conservative but modern
- relationship-driven but technically competent
- high-net-worth focused
- transparent, disciplined, and fiduciary-minded

## Visual Direction
Use Patrumin’s existing color scheme as the base:
- primary green from current site
- dark gray / charcoal
- white / off-white
- subtle light gray backgrounds
- restrained accent colors only if necessary

Design should feel closer to a private investment office than a retail financial advisor.

## Reference Design Principles
Borrow only the general design logic from:
- Morgan Stanley: clean navigation, institutional hierarchy, high-net-worth feel
- Goldman Sachs: restrained typography, white space, editorial confidence
- J.P. Morgan: conservative trust, private-client polish
- BlackRock: modern asset-management structure, clear strategy pages

## Required Site Structure
Create or improve the following sections:

1. Homepage
   - institutional hero section
   - clear value proposition
   - brief firm overview
   - investment philosophy
   - strategy preview
   - fiduciary / transparency section
   - call-to-action

2. About
   - firm history
   - investment philosophy
   - leadership / portfolio manager section

3. Investment Strategy
   - U.S. Dividends Plus+ Equity Strategy
   - disciplined equity income framing
   - suitable high-level explanation without performance promises

4. Fiduciary Audit / Portfolio Review
   - explain fee transparency
   - tax efficiency
   - legacy holdings review
   - no aggressive sales language

5. Insights
   - placeholder for market commentary

6. Contact
   - clean form or contact CTA
   - compliance-safe wording

## Design Requirements
- sticky top navigation
- responsive mobile layout
- premium hero section
- cards for strategies / services
- strong footer with disclosures
- no flashy animations
- no cartoon icons
- no generic “financial freedom” stock-photo energy
- use elegant spacing and typography
- make it feel credible to HNW and UHNW prospects

## Compliance Constraints
Avoid:
- guaranteed returns
- “beat the market”
- “safe investment”
- “risk-free”
- promissory performance language
- exaggerated claims

Use cautious language:
- “designed to”
- “seeks to”
- “focused on”
- “disciplined”
- “transparent”
- “long-term”
- “risk-aware”

## Technical Requirements
- preserve current content where useful
- modernize HTML/CSS structure
- improve accessibility
- optimize for desktop and mobile
- keep code clean and easy to maintain
- use semantic HTML
- do not introduce unnecessary frameworks unless the project already uses them

## Deliverable
Update the website files directly. After editing, explain:
1. What files changed
2. What design system was created
3. What sections were added or improved
4. Any remaining recommended next steps
